# 🚀 Productivity Dashboard

A modern and responsive **Productivity Dashboard** built using **HTML, CSS, and JavaScript(DOM)** that brings multiple productivity tools together in one place. It helps users organize tasks, manage daily schedules, track goals, focus using the Pomodoro technique, stay updated with weather and time, and get inspired through motivational quotes.

## ✨ Features

### 📋 Todo Manager
- Add new tasks
- Mark tasks as completed
- Mark Star
- Delete tasks

### 📅 Daily Planner
- Organize your daily schedule
- Time-based planning

### 🎯 Goal Tracker
- Create daily goals
- Track goal progress
- Update completion status

### ⏱️ Pomodoro Timer
- Focus timer
- Start, Pause & Reset
- Productivity tracking

### 💬 Motivational Quotes
- Random inspirational quotes
- Refresh quotes anytime

### 🌤️ Weather Widget
- Real-time weather updates
- Current temperature
- Weather condition
- City information

### 📆 Date & Time
- Live digital clock
- Current date display

### 🌙 Dark Mode
- Light/Dark theme switch

### 📱 Responsive Design
- Desktop
- Tablet
- Mobile friendly
